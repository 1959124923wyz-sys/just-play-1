# just-play-1

一个以宋朝汴梁为背景的悬疑文字冒险游戏示例。

## 运行游戏

```bash
python game.py
```

## 命令说明

- 输入 `1..N`：选择对应选项
- 输入 `0`：显示帮助
- 输入 `8`：回到上一步（起点无法后退）
- 输入 `9`：退出游戏
- 直接回车：重新显示当前节点与选项
- 其它输入：提示“无效输入”并停留在当前节点
- 结局时输入 `1` 重开，输入 `9` 退出

## 玩法简介

你将置身汴梁夜雨中的疑案，追查权力与欲望交织的真相，依据线索做出选择，触发不同结局。部分选项需要先获得线索后才会出现。

部分选项会显示为【尚不可行】，表示你尚未满足线索或银钱条件。多探索或累积人情可解锁。

数值影响被“叙事化”呈现：例如银钱增减会有叙述反馈，但不会直接显示数字。

按一次 0 显示简短帮助；连续按两次 0 可查看幕后信息（线索与数值，仅供调试）。

演示玩法：在起点选择“顺手摸走几枚铜钱”可增加银两与嫌疑；银两足够时会出现“试图买通巡卒”的选项。
