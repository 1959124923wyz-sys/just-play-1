# just-play-1

一个最小可运行的文字冒险游戏示例。

## 运行游戏

```bash
python game.py
```

## 命令说明

- 输入 `1..N`：选择对应选项
- 输入 `0`：显示帮助
- 输入 `#`：回到上一步（起点无法后退）
- 输入 `9`：退出游戏
- 直接回车：重新显示当前节点与选项
- 其它输入：提示“无效输入”并停留在当前节点
- 结局时输入 `1` 重开，输入 `9` 退出

## 运行测试

```bash
python -m unittest
```

## 故事文件

剧情在 `story.json` 中定义，启动时由 `game.py` 读取并驱动流程。
