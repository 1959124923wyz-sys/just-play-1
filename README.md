# just-play-1

一个以宋朝汴梁为背景的悬疑文字冒险游戏示例。

## 运行游戏

```bash
python game.py
```

## 命令说明

- 输入 `1..N`：选择对应选项
- 输入 `0`：显示帮助
- 输入 `8`：回到上一步（起点无法后退）
- 输入 `9`：退出游戏
- 直接回车：重新显示当前节点与选项
- 其它输入：提示“无效输入”并停留在当前节点
- 结局时输入 `1` 重开，输入 `9` 退出

## 玩法简介

你将置身汴梁夜雨中的疑案，追查权力与欲望交织的真相，依据线索做出选择，触发不同结局。部分选项需要先获得线索后才会出现。

## 运行测试

```bash
python -m unittest
```

## 故事文件

剧情在 `story.json` 中定义，启动时由 `game.py` 读取并驱动流程。
